Compile the code for 2D beach case¶

Makefile

See an example of a complete “Makefile” here.

For this example, you will need to update the EXEC variable, and make sure the appropriate FLAGS are active in the “Makefile”:

#---------BEGIN MAKEFILE-------------
   ...
   ...
   EXEC    = funwave_surface_wave          # for example

#------------------------------------
#  PRECISION ...
#
#------------------------------------
## FLAGS
## Flag numbers are arbitrary, but necessary

   FLAG_1 = -DDOUBLE_PRECISION
   FLAG_4 = -DCARTESIAN

#  if parallel add
   FLAG_2 = -DPARALLEL

#  if intel compiler add
   FLAG_6 = -DINTEL

   ...

#------------------------------------
# mpi defs
#------------------------------------
## COMPILER INFO

   ...

   FC       = mpif90       # for example

The compiled executable file that’s generated is “funwave_surface_wave.exe”.
