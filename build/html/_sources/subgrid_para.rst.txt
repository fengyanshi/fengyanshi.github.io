Subgrid Parameters
************************************

**SPECIFICATION OF TIDE AND SURGE OPEN BOUNDARY CONDITIONS** 

* :code:`SubMainGridRatio`: integer parameter for subgrid ratio

* :code:`DEPTH_SUBGRID_FILE`: string name of the file containing subgrid data


Examples can be found in /simple_cases/, or see `Subgrid module <https://fengyanshi.github.io/build/html/subgrid_module.html>`_
