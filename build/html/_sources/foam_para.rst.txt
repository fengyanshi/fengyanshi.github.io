.. _section-foam:

Parameters for Bubble and Foam 
************************************

* :code:`PLOT_INTV_FOAM`: time interval for plots of foam, default: same as PLOT\_INTERV

* :code:`MinThick`: float number, default: 0.01

* :code:`TransferRate`: float number, default: 0.1

* :code:`CdFoam`: float number, default: 0.5

Examples can be found in /simple_cases/rip\_2D\_foam.
