Regular wave 30 deg oblique incidence¶

The following sections will be modified in “input.txt” for this case. All other sections are defined in Basics for model setup for this example.

Set a descriptive title for your simulation:

!-----TITLE-----
 TITLE = 2D_beach_30deg

Modify the initial wavemaker section on Basics for model setup to generate monochromatic waves with incidence of 30.0:

WAVEMAKER = WK_REG
DEP_WK = 8.0
Xc_WK = 150.0
Yc_WK = 0.0
Tperiod = 8.0
AMP_WK = 0.5
Theta_WK = 30.0    ! this line
Delta_WK = 3.0

(refer to Wave-maker specification for description of parameters)
